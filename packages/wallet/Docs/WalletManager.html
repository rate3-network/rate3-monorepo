(async) getAccount() → {Account}

Before generating accounts, the seed and wallet MUST be there. i.e., random account generatation is NOT allowed. If the wallet is set, generate the first (0) account in the wallet; If the parameter is a number, Generate the account in the wallet. Its index is the number. If the parameter is a string, Use it as the private key to generate the account
